# Map Management

Users can create, list, open, and delete thesis maps.

## Overview

A user can have multiple thesis maps. Each map represents one thesis statement and its supporting/opposing evidence tree.

## Map Properties

- **Name / title** â€” A human-readable label for the map

## Landing Page

- The landing page displays a list of all existing maps
- Each entry shows the map name
- The user can open a map to view and edit it
- The user can delete a map from the list

## Creating a Map

- The user can create a new map from the landing page
- Creating a map prompts the user for a thesis statement
- The thesis statement becomes the root node of the new map

## Deleting a Map

- The user can delete a map
- Deleting a map removes the map and all of its nodes, groups, and attachments
